Wide SPFX Workbench

Description

This extension automatically maximizes the width of your SharePoint Framework (SPFX) workbench by removing layout restrictions that limit the canvas to 924px width.

Features

  • Automatically detects SPFX workbench pages
  • Removes max-width restrictions (extends to 2560px)
  • Hides unnecessary UI elements
  • Removes padding constraints
  • Works automatically on page load and navigation

How it Works

The extension monitors your browser tabs and automatically applies width modifications when you visit SharePoint workbench pages (/_layouts/15/workbench.aspx). No manual intervention required!

Version

v1.0 - Initial Release
